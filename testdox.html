Base Url Http Client (CoiSA\Http\Client\BaseUrlHttpClient)

  • ✓ Class implements http client interface
  • ✓ Construct with string base url will create new base uri factory
  • ✓ Get base url request factory will return base url request factory with given base uri factory
  • ✓ Create request will proxy create request to base url request factory
  • ✓ Send request will recreate uri and proxy send request to given client
